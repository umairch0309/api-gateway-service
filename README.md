# Api Gateway Service

A microservices API gateway service built with Node.js/Express and TypeScript for routing and managing API requests.

## Tech Stack

- **Runtime**: Node.js
- **Framework**: Express.js
- **Language**: TypeScript/JavaScript
- **Package Manager**: npm

## Project Structure

```
api-gateway-service/
├── src/
│   ├── controllers/
│   ├── models/
│   ├── routes/
│   ├── middleware/
│   └── utils/
├── tests/
├── .env.example
├── package.json
├── tsconfig.json
└── README.md
```

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Copy environment variables:
   ```bash
   cp .env.example .env
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
